01

Présentation de JavaScript

JavaScript est le langage du comportement côté client. Il permet de rendre une page vivante, d'interagir avec l'utilisateur, de manipuler le DOM et d'échanger des données avec un serveur.

02

Inclusion du JavaScript

Ici, le script est inclus avec <script src="app.js" defer></script> afin d'attendre le chargement du HTML avant l'exécution de la logique applicative.
