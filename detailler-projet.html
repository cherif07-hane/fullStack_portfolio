Projet Portfolio

Cette page présente un projet développé autour d'une interface portfolio simple, structurée, lisible et pensée pour mettre en avant le contenu.

Description

Le projet met en avant plusieurs pages HTML reliées par une navigation commune. Le style a été refactorisé avec Tailwind CSS pour obtenir une meilleure cohérence visuelle, des bordures mieux maîtrisées et une structure responsive plus propre.
